# 题目描述
给定一个非空整数数组，除了某个元素只出现一次以外，其余每个元素均出现两次。找出那个只出现了一次的元素。

说明：
你的算法应该具有线性时间复杂度。 你可以不使用额外空间来实现吗？
```
示例 1:
输入: [2,2,1]
输出: 1
示例 2:
输入: [4,1,2,1,2]
输出: 4
```
# 思路
异或运算满足以下定律：（离散数学有详解）
交换律：a ^ b ^ c <=> a ^ c ^ b
任何数与0异或 0 ^ n = n
相同的数异或为0 n ^ n => 0
根据三条定律，用0与数组中所有元素分别异或，最后会得到没有相同的那个数。

##代码：
```c++
class Solution {
public:
    int singleNumber(vector<int>& nums) {
            
        int ret = nums[0];
        
        for(int i = 1; i<nums.size(); i++)
        {
            ret ^= nums[i];
        }
        
        return ret;
        
    }
};
```
